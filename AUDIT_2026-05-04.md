## Top-3-Empfehlung für nächste Runde

### #1 — Phase 3 echt machen — Kunden-Bridges + Kommunikations-Log (0.5 Session)
**Warum:** Stellenbeschreibung sagt „Kunde im Mittelpunkt" — Tool hat null Kunden-Kontakt-Mechanismus. Kleinster Eingriff, maximaler Realwert.
**Was:** 3 neue S-Felder (kundenTel/kundenWhatsApp/kundenEmail) + 3 Hero-Bridges (📱✉📞) + Ergebnis-Inline-Form nach Klick.

### #2 — Tier 4.3 Pipeline-Bidi (1 Session)
**Warum:** Phase 2 ist das größte Loch. Mit Multi-Order jetzt sinnvoll. Eliminiert die häufigste Fehlerquelle (Drift Tool ↔ Pipeline).
**Was:** CSV-Import mit Drift-Detector + Export-Zeile (Copy-Paste) pro Auftrag + Sync-Vorbereitungs-Übersicht.

### #3 — Tier 2.1b Bucket-spezifische Sub-Flows (1–2 Sessions)
**Warum:** Holt die größte Strukturreserve aus dem 7-Bucket-Display. Jeder Bucket bekommt die richtigen Pflichtfelder. Erst damit wird das Cockpit ein Cockpit.
**Was:** pro Bucket eigene Inhalts-Card mit Pflichtfeldern + Bucket-Lock-Logic (Wechsel gesperrt bis Pflichten erfüllt).
